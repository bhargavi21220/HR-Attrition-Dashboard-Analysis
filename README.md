# HR-Attrition-Dashboard-Analysis

## 📌 Project Overview

The HR Attrition Analysis Dashboard is an interactive data analytics project developed to analyze employee attrition trends and workforce insights. The dashboard helps identify key factors influencing employee turnover, including job satisfaction, overtime, income, age, department, and job role.

Built using Python, Pandas, Plotly, and Streamlit, this project provides interactive visualizations and business recommendations to support data-driven HR decision-making.

## 📈 Key Insights

* Employees working overtime show higher attrition rates.
* Job satisfaction significantly impacts employee retention.
* Certain departments and job roles experience higher employee turnover.
* Younger employees demonstrate higher attrition tendencies.
* Income and work-life balance influence retention outcomes.
